# Todo List App

A clean and responsive Todo List application developed using HTML, CSS, and Vanilla JavaScript.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks with smooth animation
- Filter tasks (All, Completed, Uncompleted)
- Save tasks using Local Storage
- Responsive and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
- Font Awesome

## Project Preview
A simple productivity application that helps users organize daily tasks while demonstrating DOM manipulation, event handling, and browser storage in JavaScript.
